# 화면기획서: S-08 운동 완료 화면

---

## 📌 개요

* **화면명**: 운동 완료
* **목적**: 사용자의 운동 종료를 축하하고 다음 액션으로 안내한다

---

## 📂 구성 요소

* **칭찬 문구 영역**:

  * 무작위로 선택되어 출력되는 기분 좋은 문구
  * 예시:

    * "와! 오늘도 완벽하게 해냈어요 💪"
    * "몸이 기억할 거예요, 이 노력! ✨"
    * "하루 한 걸음, 멋진 당신의 루틴!"
    * "대단해요! 자신과의 약속을 지켰어요."
    * "운동 완료! 이젠 쉬어도 좋아요 😊"

* **하단 버튼**:

  * \[운동 마치기] (Primary)

    * 클릭 시 오늘 운동 요약 화면으로 이동 (S-09)
  * \[다른 운동하기] (Secondary)

    * 클릭 시 프리셋 선택 화면 (S-02)으로 이동

---

## 🔁 동작

* 운동 종료 시 자동 진입
* 버튼 클릭에 따라 다음 화면으로 분기
