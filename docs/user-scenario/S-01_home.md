# 화면기획서: S-01 홈 화면

---

## 📌 개요

* **화면명**: 홈
* **목적**: 사용자에게 운동 시작 진입점을 제공하고, 앱의 시작점 역할 수행

---

## 📂 구성 요소

* **타이틀 영역 (앱명 또는 로고)**

* **메인 버튼**:

  * \[운동 시작] (primary)

    * 클릭 시 S-02 프리셋 선택 화면으로 이동

* **로그인 버튼**:
  * \[구글 로그인]
    * '운동 시작' 버튼 하단에 위치
    * 클릭 시 구글 로그인 플로우 실행

---

## 🔁 동작

* 앱 실행 시 가장 먼저 진입하는 화면
* \[운동 시작] 클릭 시 ➝ S-02 프리셋 선택 화면으로 이동
* \[구글 로그인] 클릭 시 ➝ 구글 계정 선택 및 로그인 인증 수행
