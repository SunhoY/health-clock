# 화면기획서: S-08 운동 완료 화면

---

## 📌 개요

* **화면명**: 운동 완료
* **목적**: 운동 종료를 짧게 피드백하고 다음 행동으로 연결한다

---

## 📂 구성 요소

* **완료 메시지 영역**:

  * 짧은 문구 1개 노출 (랜덤)
  * 예시: "운동 완료", "좋아요, 끝냈어요", "오늘도 성공"

* **하단 버튼**:

  * `[운동 마치기]` (Primary) ➝ S-09 이동
  * `[다른 운동]` (Secondary) ➝ S-02 이동

* **모바일 UI 기준**:

  * 완료 메시지는 최대 1줄
  * Primary 버튼을 하단 고정
  * 감성 문구는 짧고 즉시 이해 가능해야 함

---

## 🔁 동작

* 운동 종료 시 자동 진입
* 버튼 클릭에 따라 다음 화면 분기
