# 화면기획서: S-08 운동 완료 화면

---

## 📌 개요

* **화면명**: 운동 완료
* **목적**: 운동 종료 직후 핵심 결과만 빠르게 확인하고 종료/상세 확인으로 분기한다

---

## 📂 구성 요소

* **완료 메시지 영역**:

  * 고정 문구 1개 노출
  * 예시: `운동 끝!`

* **핵심 요약 영역**:

  * 오늘 운동 시간
  * 완료한 운동 목록 (운동명 단위)
  * 제외 정보:
    * 총중량
    * 총 세트수
    * 감성 문구/설명 문장 (`오늘의 운동 요약` 같은 제목 포함)

* **하단 버튼**:

  * `[운동 마치기]` (Primary) ➝ S-01 이동
  * `[상세 요약 보기]` (Secondary) ➝ S-09 이동

* **모바일 UI 기준**:

  * 완료 메시지는 최대 1줄 (`운동 끝!`)
  * 요약 정보는 2개 블록(운동 시간/운동 목록)만 표시
  * Primary 버튼을 하단 고정
  * 반복 진입 시에도 과한 정보 없이 즉시 종료 가능해야 함

---

## 🔁 동작

* 운동 종료 시 자동 진입
* 기본 화면은 간소 요약만 노출
* `[운동 마치기]` 클릭 시 홈(S-01)으로 이동
* `[상세 요약 보기]` 클릭 시 상세 화면(S-09)으로 이동
