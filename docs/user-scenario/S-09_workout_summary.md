# 화면기획서: S-09 운동 상세 요약 화면

---

## 📌 개요

* **화면명**: 운동 상세 요약
* **목적**: 오늘 수행한 운동을 부위 단위로 상세 요약해 확인하게 한다

---

## 📂 구성 요소

* **상단 영역**:

  * `오늘 운동 부위` 표시
  * 오늘 수행한 부위만 출력 (`유산소 / 상체 / 하체 / 코어`)

* **부위별 상세 카드**:

  * `유산소` 카드
    * `소모한 칼로리` 표시
  * `상체`, `하체`, `코어` 카드
    * `총 세트수`
    * `최대 중량`

* **조건부 출력 규칙**:

  * 오늘 수행하지 않은 부위 카드는 출력하지 않음
  * 예: 오늘 상체만 수행했다면 `상체` 카드만 표시

* **하단 버튼**:

  * `[홈으로]` (Primary)

* **모바일 UI 기준**:

  * 부위 카드는 스크롤로 빠르게 훑을 수 있게 간결하게 유지
  * 불필요한 설명 텍스트/감성 카피 제거
  * 핵심 행동은 "상세 확인 후 종료" 한 가지로 유지

---

## 🔁 동작

* S-08에서 `[상세 요약 보기]` 클릭 시 진입
* 해당 세션/오늘 데이터 기반으로 부위별 뷰모델 생성 후 렌더링
* `[홈으로]` 클릭 시 S-01 홈 화면 이동
