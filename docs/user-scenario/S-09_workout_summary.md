# 화면기획서: S-09 운동 상세 요약 화면

---

## 📌 개요

* **화면명**: 운동 상세 요약
* **목적**: 필요할 때만 상세 운동 기록을 확인하도록 보조한다

---

## 📂 구성 요소

* **상세 리스트**:

  * 웨이트: `{운동} · {세트수}세트`
  * 유산소: `{운동} · {시간}분`

* **보조 정보**:

  * MVP 단계에서는 칼로리/근성장 추정치 제외

* **하단 버튼**:

  * `[홈으로]` (Primary)

* **모바일 UI 기준**:

  * 부위/운동/수치를 한 줄 요약으로 표시
  * 긴 설명 없이 리스트 스캔이 가능해야 함
  * 핵심 행동은 "요약 확인 후 종료" 한 가지로 유지

---

## 🔁 동작

* S-08에서 `[상세 요약 보기]` 클릭 시 진입
* 해당 세션 운동 내역 기반으로 리스트 동적 생성
* `[홈으로]` 클릭 시 S-01 홈 화면 이동
