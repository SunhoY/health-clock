# 화면기획서: S-02 프리셋 선택 화면

---

## 📌 개요

* **화면명**: 프리셋 선택
* **목적**: 사용자가 저장된 루틴(프리셋)을 선택하거나 새로운 루틴을 만들 수 있도록 한다

---

## 📂 구성 요소

* **프리셋 리스트**:

  * 저장된 프리셋 목록 카드 또는 리스트 형태
  * 각 항목 클릭 시 해당 운동 루틴으로 진입 (S-07 운동 진행 화면으로 이동)

* **빈 상태 화면** (프리셋이 없을 경우):

  * "아직 저장된 루틴이 없어요."
  * \[운동 추가하기] 버튼 ➝ S-03 루틴 만들기 플로우 진입

* **하단 고정 버튼**:

  * \[운동 추가하기]

    * 클릭 시 S-03 루틴 만들기 - 부위 선택 화면으로 이동

---

## 🔁 동작

* 기존 프리셋 선택 시 ➝ 운동 바로 시작
* \[운동 추가하기] 클릭 시 ➝ 루틴 만들기 플로우 시작
