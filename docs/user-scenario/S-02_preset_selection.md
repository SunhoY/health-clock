# 화면기획서: S-02 프리셋 선택 화면

---

## 📌 개요

* **화면명**: 프리셋 선택
* **목적**: 저장된 루틴을 빠르게 선택하거나 새 루틴을 추가하게 한다

---

## 📂 구성 요소

* **상단 제목**: 프리셋 선택
* **프리셋 리스트**:

  * 카드 또는 리스트 형태
  * 각 항목 탭 시 S-07 운동 진행 화면으로 이동

* **빈 상태 화면** (프리셋이 없을 경우):

  * 안내 문구: 저장된 루틴이 없어요.
  * `[루틴 만들기]` 버튼 ➝ S-03 이동

* **하단 고정 버튼 (Primary)**:

  * `[루틴 만들기]`
  * 클릭 시 S-03 루틴 만들기 플로우 진입

* **모바일 UI 기준**:

  * 리스트 항목 전체를 탭 가능 영역으로 제공
  * CTA 문구 통일: "운동 추가하기" 대신 "루틴 만들기"
  * 한 화면 핵심 행동은 "프리셋 선택"으로 유지

---

## 🔁 동작

* 기존 프리셋 선택 시 ➝ 운동 바로 시작
* `[루틴 만들기]` 클릭 시 ➝ S-03 이동
