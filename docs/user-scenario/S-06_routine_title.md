# 화면기획서: S-06 루틴 저장 - 제목 입력 팝업

---

## 📌 개요

* **화면명**: 제목 입력 팝업
* **목적**: 루틴 저장 전 제목을 간단히 확인/수정하게 한다

---

## 📂 구성 요소

* **입력 필드**:

  * 기본값: 마지막 선택 부위명 (예: 가슴)
  * 최대 길이: 20자
  * 플레이스홀더: 루틴 이름 입력

* **버튼**:

  * `[저장]` (Primary): 제목 저장 후 S-02 이동
  * `[취소]` (Secondary): 팝업 닫기

* **모바일 UI 기준**:

  * 팝업 내 설명 문구 최소화
  * 자동 포커스로 키보드 즉시 표시
  * 저장 버튼은 키보드 노출 상태에서도 가려지지 않게 배치

---

## 🔁 동작

* `[저장]` 클릭 시 ➝ 프리셋 저장 및 S-02 프리셋 선택 화면 이동
* `[취소]` 클릭 시 ➝ S-05로 복귀
