# 화면기획서: S-06 루틴 저장 - 제목 입력 다이얼로그

---

## 📌 개요

* **화면명**: 제목 입력 다이얼로그
* **목적**: 루틴 저장 전 제목을 간단히 입력하고 저장하게 한다

---

## 📂 구성 요소

* **입력 필드**:

  * 초기값(value): 빈 값
  * 최대 길이: 50자
  * 플레이스홀더: 추천 제목(예: 가슴 루틴, 복합 루틴)

* **버튼**:

  * `[저장]` (Primary): 제목 저장 후 S-02 이동
  * `[취소]` (Secondary): 다이얼로그 닫기

* **모바일 UI 기준**:

  * 다이얼로그 내 설명 문구 최소화
  * 자동 포커스로 키보드 즉시 표시
  * 저장 버튼은 키보드 노출 상태에서도 가려지지 않게 배치

---

## 🔁 동작

* S-05에서 `[완료]` 클릭 시 `/routine-title` 진입과 함께 제목 입력 다이얼로그를 표시한다
* 입력값이 없으면 placeholder만 보이고 저장되지 않는다
* `[저장]` 클릭 시 실행 모드별 저장 경로를 사용한다
  * 인증 모드: 루틴 생성 API 호출 후 성공 시 S-02 이동
  * 게스트 모드: 로컬 스토리지 저장 후 S-02 이동
* S-02 복귀 직후 실행 모드 저장소 재조회 결과에 방금 추가한 루틴이 노출되어야 한다
* `[취소]` 클릭 시 ➝ 다이얼로그를 닫고 S-05로 복귀
