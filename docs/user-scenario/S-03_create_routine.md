# 화면기획서: S-03 루틴 만들기 - 부위 선택 화면

---

## 📌 개요

* **화면명**: 루틴 만들기 - 부위 선택
* **목적**: 루틴 생성 첫 단계에서 운동 부위를 빠르게 선택하게 한다

---

## 📂 구성 요소

* **제목**: 루틴 만들기
* **보조 문구**: 운동 부위를 선택하세요.
* **운동 부위 선택 버튼 (2열 그리드 권장)**:

  * 가슴
  * 등
  * 하체
  * 어깨
  * 팔
  * 코어(복부)
  * 종아리
  * 전신
  * 유산소

* **부위 구성 검토 (MVP 범위)**:

  * 현재 지원 부위: 가슴, 등, 하체, 어깨, 팔, 코어(복부), 종아리, 전신, 유산소
  * 누락 우려 부위는 MVP에서 다음으로 포함 처리:
    * 엉덩이(둔근) ➝ 하체/전신
    * 전완/손목 ➝ 팔
    * 목 ➝ 전신

* **모바일 UI 기준**:

  * 버튼 간 간격 8px 이상
  * 각 버튼 터치 영역 최소 44x44px
  * 스크롤 시에도 제목과 현재 단계가 인지되도록 구성

---

## 🔁 동작

* 부위 버튼 클릭 시 ➝ S-04 세부 운동 선택 화면 이동
* S-02에서 `[편집]`으로 진입한 경우(`mode=edit`, `presetId` 전달)에는 본 화면을 표시하지 않고 S-04 편집 모드로 즉시 이동한다
* 운동 부위 목록은 인증 모드/게스트 모드 모두 서버 API(`GET /api/exercises/body-parts`) 조회 결과를 사용한다
