# 헬스클록 기획 문서

---

## 1. 제품 개요

* **제품명**: 헬스클록 (가칭)
* **목표**: 사용자가 운동 중 최소한의 조작으로 운동 종류, 세트 수, 중량, 휴식 등을 기록하고 트래킹할 수 있도록 한다.
* **주 사용자**: 기획자 본인 (개인 트레이닝용)
* **핵심 가치**: 간편한 UX, 빠른 입력, 자동화된 흐름

---

## 2. 주요 기능 요약

| 기능 ID | 기능 설명                                  |
| ----- | -------------------------------------- |
| F-01  | 부위 선택 (가슴, 등, 하체, 유산소 등)               |
| F-02  | 운동 프리셋 선택 및 생성 기능                      |
| F-03  | 부위 선택 시 해당 부위 운동 목록 표시                 |
| F-04  | 운동별 세트 수, 중량, 시간 입력 (스피너 방식)           |
| F-05  | 세트 간 휴식 시간 자동 제공 및 진행 상태 표시            |
| F-06  | 운동 추가/편집 시 운동 리스트를 구성하여 프리셋으로 저장       |
| F-07  | 운동 진행 화면에서 세트 수행 → 휴식 → 다음 세트 흐름 제공    |
| F-08  | 모든 운동 완료 시 기분 좋은 응원 문구 + 기록 저장 + 요약 제공 |

---

## 3. 사용자 흐름 (User Flow)

```plaintext
홈 화면
 └─ 운동 시작
      ├─ 프리셋 선택 → 운동 진행
      └─ 프리셋 없음 → 루틴 만들기(부위 선택 → 운동 선택 → 세트 구성 반복)
            └─ 제목 입력 후 프리셋 저장 → 운동 순서 설정 (준비 중)
                 └─ 운동 시작
                      └─ [운동 화면: 세트 ↔ 휴식 반복]
                           └─ 완료 → 운동 완료 화면
                                └─ 기록 저장 및 요약 확인
```

---

## 4. 화면 구성 기획

### S-01 홈 화면

* 앱 타이틀
* \[운동 시작] 버튼

### S-02 프리셋 선택 화면

* 기존 루틴 리스트 (프리셋)
* \[운동 추가] 버튼 (프리셋 없을 경우 → 루틴 만들기 플로우 진입)

### S-03 루틴 만들기: 부위 선택 화면

* 제목: 루틴 만들기
* 부제: 운동 부위를 선택하세요.
* 부위 버튼 리스트 (가슴, 등, 하체, 어깨, 팔, 복부, 종아리, 전신, 유산소)

### S-04 루틴 만들기: 세부 운동 선택 화면

* 제목: \[선택한 부위] 운동
* 운동 버튼 리스트 (예: 벤치프레스, 데드리프트 등)

### S-05 루틴 만들기: 세부항목 입력 화면

* 제목: \[선택한 운동명]
* 세트 수: 1씩 증감 스피너
* 중량 (kg): 5씩 증감 스피너
* 시간 (분): 1씩 증감 스피너
* 하단 버튼: \[운동 추가하기], \[끝내기]

### S-06 루틴 저장 화면 (팝업)

* 제목 입력 필드 (기본값: 선택한 부위명)
* 확인 시 프리셋 저장 후 프리셋 화면으로 복귀

### S-07 운동 진행 화면

* 현재 운동 정보 (운동명, 세트 진행 상황)
* \[세트 완료] 버튼 → 휴식 진행
* \[휴식 건너뛰기], \[그만두기] 버튼

### S-08 운동 완료 화면

* 칭찬 메시지 (랜덤)
* 버튼: \[운동 마치기] (→ 요약 화면), \[다른 운동하기] (→ 프리셋 화면)

### S-09 운동 기록 요약 화면

* 오늘의 운동 리스트 요약

  * 웨이트: {부위} - {운동} - {세트수}세트
  * 유산소: 유산소 - {운동} - {시간}분
* 소모 칼로리, 근성장 정보는 MVP에서는 표시하지 않음

## 6. 향후 추가 기능 (메모)

* 운동별 칼로리 소모량 계산 (중량, 횟수, 세트 기반 예측)
* 유튜브 운동 영상 추천
* 신체 조건 기반 운동 추천 (몸무게, 키 등 고려)
* 운동 순서 설정 화면 (drag & drop)
* 운동 중 음성 피드백
* 프리셋 공유 기능 (URL 또는 QR)
