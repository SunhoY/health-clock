# 화면기획서: S-06 루틴 저장 - 제목 입력 팝업

---

## 📌 개요

* **화면명**: 제목 입력 팝업
* **목적**: 운동 구성이 완료된 후 사용자가 프리셋 제목을 설정할 수 있도록 한다

---

## 📂 구성 요소

* **입력 필드**:

  * 기본값: 마지막에 선택된 부위명 (예: "가슴")
  * 사용자가 자유롭게 수정 가능

* **버튼**:

  * \[확인]: 제목 저장 및 프리셋 저장 → 프리셋 선택 화면으로 복귀

---

## 🔁 동작

* \[확인] 클릭 시 ➝ 프리셋 저장 및 S-02 프리셋 선택 화면으로 이동
